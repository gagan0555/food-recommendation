# Food Recommendation System

## Overview
A community-driven food recommendation platform built with React, Node.js, and MongoDB. It helps users discover authentic local food spots through questions and reviews.

## Features
- User authentication and profile management
- Food recommendations and question posting
- Responsive UI with React and Tailwind CSS
- Backend API with Express and MongoDB
- Cloud deployment on Render
